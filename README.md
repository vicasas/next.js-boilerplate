<h1 align="center">Next.js Boilerplate</h1>

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app), includes the essential tools needed for an optimal development experience.

## Features

- 🚀 **[TypeScript](https://www.typescriptlang.org/)**: Type-checking for enhanced productivity.
- 💅 **[Eslint](https://eslint.org/)**: Consistent code style & error detection.
- 🚨 **[Prettier](https://prettier.io/)**: Automatic code formatting.
- 🐭 **[Editorconfig](https://editorconfig.org/)**: Consistent coding styles.
- 🧪 **[Jest](https://jestjs.io/), [React Testing Library](https://testing-library.com/react/)**: Unit & integration testing.
- 🎭 **[Playwright](https://playwright.dev/)**: End-to-end testing.
- 🚫 **[Lint Staged](https://github.com/okonet/lint-staged)**: Pre-commit linting.
- 🐶 **[Husky](https://typicode.github.io/husky/#/)**: Git hooks for code quality.
- 📊 **[Bundle Analyzer](https://www.npmjs.com/package/webpack-bundle-analyzer)**: Webpack bundle insights.
- 🔄 **[Renovate](https://renovatebot.com/)**: Dependency updates automation.
- ⚙️ **[VSCode Settings](https://code.visualstudio.com/docs/getstarted/settings)**: VSCode presets.
- ⬆️ **[NVM File](https://github.com/nvm-sh/nvm)**: Node.js version control.
- 🔧 **[GitHub Actions](https://github.com/features/actions)**: CI/CD workflows automation.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
